# MonsterBox简易仿制版
A simple copy version MonsterBox developed with HTML, CSS, and JavaScript.

## 功能
- 玩家通过方向键（↑↓←→）控制角色移动，Z 键释放攻击
- 击败敌人可获取金币，积累金币可强化角色属性（生命值、攻击力、攻击范围等）
- 敌人会自动追击玩家，玩家受击后获得短暂无敌时间
- 通关后解锁属性加点界面，敌人难度随通关次数递增
- 玩家生命值归零时游戏结束，清空当前敌人可进入下一轮挑战
- 边界碰撞限制（角色无法超出画布范围）

## 如何运行
直接打开保存游戏代码的 HTML 文件，即可在浏览器中自动启动游戏，按空格键开始第一局。

## Features
- Players control the character with arrow keys (↑↓←→) and release attacks with the Z key
- Defeat enemies to earn gold, which can be used to enhance character attributes (health, attack power, attack range, etc.)
- Enemies automatically chase players, and players gain a short period of invincibility after taking damage
- Unlock the attribute upgrade interface after clearing a level; enemy difficulty increases with each cleared level
- Game over when the player's health drops to zero; clear all enemies to enter the next challenge
- Boundary collision restriction (the character cannot move beyond the canvas range)

## How to Run
Simply open the HTML file where the game code is saved, and the game will start automatically in the browser. Press the spacebar to begin the first round.
